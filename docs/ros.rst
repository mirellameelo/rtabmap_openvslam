==============
Running on ROS
==============

We provide ROS and ROS2 package examples to help you run OpenVSLAM on ROS framework.

.. NOTE ::

    Please build OpenVSLAM with **OpenCV 3.3.1 or later** if you plan on using ROS package.
    OpenCV 4.x is not supported yet.
